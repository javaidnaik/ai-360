This page will help you debug your environment variable configuration.

1. Check Browser Console

Open your browser's developer tools (F12) and check the Console tab for debug messages.

2. Common Issues & Solutions

Issue: Environment variables not loading
  • Make sure your .env.local file is in the root directory (same level as package.json)
  • Restart your dev server after creating/updating .env.local
  • Environment variables must start with VITE_ to be available in the browser
Issue: "Failed to construct 'URL': Invalid URL"
  • The VITE_SUPABASE_URL is empty or invalid
  • Check that your .env.local file has the correct format
  • Make sure there are no quotes around the values
